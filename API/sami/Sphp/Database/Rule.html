static Rule binaryOperationCompare(string|Query $column, string $binOp, string|int|bool $value, string $op, string|int|bool $result)

Generates a rule a bitwise operation to the column value pair and compares the result to a given parameter

NOTE! This method quotes automatically $value, $result inputs!

Values for parameter $binOp:

  • &: bitwise AND
  • |, bitwise OR
  • ^: bitwise XOR
  • ~: invert bits
  • <<: left shift
  • >>: right shift

Parameters

string|Query $column the column
string $binOp Bitwise operand
string|int|bool $value the value to bitwise compare to
string $op logical operation
string|int|bool $result the result value of the bitwise operation
