Config

class Config implements Arrayable, Iterator, ArrayAccess, Countable

Application Configuration class for storing common application data

Note: Uses Singleton pattern

Stored variable names can either be an integer or a string. Additionally the following $varName casting is equal with the PHP array key casting. The $value can be of any type.
