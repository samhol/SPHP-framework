static string implode(array $arr, string $glue = '', string $lastGlue = null)

Implodes all elements of an (optionally multidimensional) array with a string

Notes:

  • Supports multidimensional arrays.
  • Returns a string containing a string representation of all the array elements in the same order, with the $glue string between each element.
  • Inserts the optional $lastGlue string between the last two elements if given. (Otherwise uses the $glue string)

Parameters

array $arr multidimensional array of strings to implode
string $glue string between each array element
string $lastGlue optional string between the last two array elements

Return Value

string the imploded array
