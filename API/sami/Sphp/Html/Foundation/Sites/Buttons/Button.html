$this setColor(string|null $style = null)

Sets the color (a CSS class)

Predefined values of $style parameter:

  • null unsets all special button styles (default)
  • 'alert' for alert/error buttons
  • 'success' for ok/success buttons
  • 'info' for information buttons
  • 'secondary' for alternatively styled buttons
  • 'disabled' for disabled buttons

Parameters

string|null $style one of the CSS class names defining button styles

Return Value

$this setSize(string|null $size = null)

Sets the size of the button

Predefined values of $size parameter:

  • 'tiny' for tiny buttons
  • 'small' for small buttons
  • 'medium' for "medium" (default) buttons
  • 'large' for large buttons
  • 'extend' for extended buttons (takes the full width of the container)

Parameters

string|null $size CSS class name defining button size. null value corresponds to no explicit size definition.
