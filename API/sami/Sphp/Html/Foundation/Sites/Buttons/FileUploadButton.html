FileUploadButton

class FileUploadButton implements ContentInterface, IdentifiableInputInterface, ButtonInterface

Implements <input type="submit"> tag as a Foundation Button

A submit button is used to send form data to a server. The data is sent to the page specified in the form's action attribute.
