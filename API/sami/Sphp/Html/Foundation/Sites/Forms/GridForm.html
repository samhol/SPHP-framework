GridForm

class GridForm extends AbstractContainerComponent implements IteratorAggregate, GridInterface, TraversableFormInterface

Implements a framework form

A {@link GridForm} is built with a combination of standard form elements, as well as the Foundation Grid ({@link Row}(s) and {@link Column}(s)).

$this setAttr(string $name, mixed $value = null)

Sets an attribute name value pair

IMPORTANT!: Does not alter locked attribute values:

  1. For 'class' attribute: if a CSS class name is locked the method does nothing
  2. For any other locked attribute the method throws a {@link LockingException}

$value handling:

  1. the type of the value should always convert to string
  2. null or an empty string: an empty attribute is set
  3. boolean true: an empty attribute is set
  4. boolean false: attribute is removed if present
  5. otherwise the attribute value the string conversion value

Parameters

string $name the name of the attribute
mixed $value the value of the attribute

Return Value

$this setTarget(string $target)

Sets the value of the target attribute

Notes:

The target attribute specifies a name or a keyword that indicates where to display the response that is received after submitting the form. The target attribute defines a name of, or keyword for, a browsing context (e.g. tab, window, or inline frame).

$target values:

  • _blank: The response is displayed in a new window or tab
  • _self: The response is displayed in the same frame (this is default)
  • _parent: The response is displayed in the parent frame
  • _top: The response is displayed in the full body of the window
  • framename: The response is displayed in a named iframe

Parameters

string $target the value of the target attribute

Return Value
