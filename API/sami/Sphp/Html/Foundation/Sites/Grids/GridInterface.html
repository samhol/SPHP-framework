Prepends a new row to the grid

Important!

  • $row not extending {@link RowInterface} is wrapped inside a {@link RowInterface} component using {@link self::toRow()} method.
  • The numeric keys of the content will be renumbered starting from zero and the index of the prepended row is 'int(0)'
