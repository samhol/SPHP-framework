ResponsiveEmbed

class ResponsiveEmbed extends AbstractComponent implements ResponsiveEmbedInterface, LazyMedia

Implements a Responsive embed component

This component lets browsers automatically scale iframe objects in webpages. If a video is embedded from YouTube, Vimeo, or another site that uses iframe, embed or object elements, video can be wrap into {@link self} to create an intrinsic ratio that will properly scale the iframe based media on any device.
