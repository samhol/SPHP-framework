The possible login actions:

  1. logout (happen when user clicks logout button)
  2. login via $_REQUEST data, which means simply logging in via the login form. after the user has submit his username/password successfully, his logged-in-status is written into his session data on the server. this is the typical behaviour of common login scripts.
