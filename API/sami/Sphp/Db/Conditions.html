Conditions compare(string $column, string $operator, mixed $expr = null)

Returns a new condition to compare a column to given value(s)

NOTE!
ALWAYS SANITIZE ALL VALUES THAT ARE FROM USER INPUTS!

Values for parameter $operator:

  • =: equal
  • !=, <>: not equal
  • <: less than
  • <=: less than or equal
  • >: greater than
  • >=: greater than or equal
  • LIKE: $column is like the string pattern in the $expr
  • NOT LIKE: $column is not like the pattern string given as $expr
  • IN: $column is in the the group given in $expr

  • NOT IN: $column is not in the the group given in $expr

    For $operator values LIKE and NOT LIKE use the % sign to define wildcards (missing letters in the pattern). The % sign can be used both before and after the pattern.

Parameters

string $column the column
string $operator used comparison operator
mixed $expr the value

Return Value
